## Cooldown Rules

### When to add a theme to cooldowns:
- Theme attempted 2+ times in 14 days → add to active cooldowns
- "Attempted" includes: published, rejected, discarded, banked

### Cooldown duration:
- Standard: 3 weeks from last attempt
- Heavy use (3+ attempts): 4-6 weeks
- Permanently blocked examples: indefinite (see instructions.md)

### How to check:
1. Before generating, scan this file
2. If theme is in Active Cooldowns → hard reject, generate from different territory
3. If theme was recently published → check "Recently Published" for minimum spacing
